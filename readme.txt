HOW TO EXECUTE FILE
1. Run this in Python 3.10 or 3.11 . Some certain modules cannot run in 3.12 .
2. Run "pip install -r requirements.txt" in the terminal.
3. Replace "PLACE YOUR OPENAI KEY HERE" in line 8 of app.py with your own OpenAI key.
4. Run app.py
5) Open localhost:5000 on browser

Bonus:
- Change the sys_message content in line 21 of app.py to anything of your liking.
